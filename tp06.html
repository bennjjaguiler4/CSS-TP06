1.1- ¿Qué significa CSS?

CSS significa Hojas de Estilo en Cascada (Cascading Style Sheets)

1.2- ¿Para qué sirve el CSS? ¿Qué programa se utiliza para interpretar el CSS? (es para )

css sirve para que un elemento html, tenga el resultado visual esperado (define aspectos visuales de un sitio, crea caracteristicas en general de un sitio web, y es facil de entender), el programa que se utiliza para interpretar css es el navegador.

1.3- ¿A qué se llama regla de CSS?

se le llama reglas a los conjuntos de propiedades y valores que aplicamos a una o varias etiquetas segun necesitemos.

1.4- ¿Cuál es la anatomía de la regla de CSS? Agregar imagen explicativa y utilizando una lista descriptiva explicar sus elementos.

la regla esta compuesta por los siguientes elementos

selector-selecciona los elementos a dar estilo
declaración- aplica las propiedades que querés dar al elemento seleccionado en el Selector
propiedades- es lo que te permite dar estilo a un elemento html
valor de la propiedad- las propiedades deben tener un valor. Cada propiedad tiene valores definidos

1.5- ¿Qué es un selector en CSS?

un selector en css es la primera parte de una regla de css, es un patron de elementos que le indicia al navegador que elementos html deben ser estilizados

1.6- Identificar y definir los selectores: de ID, de clases, universal, de etiquetas, descendente.

selector de id: este atributo se aplica a un unico elemento, identifica un elemento html por medio de un nombre.
selector de clases: este atributo empieza siempre por un punto que los clientes web distinguen,aplica elementos css a elementos similares entre si.
selector universal: nos deja aplicar estilos a todos los elementos html de un documento, y se indica con el asterisco (*)
selector de etiquetas: aplica propiedades CSS a una etiqueta, La misma se puede identificaqr sin utilizar simbolos de mayor y menor
selector descendente: con este selector seleccionamos un elemento que está dentro de otro del mismo tipo, siempre esta formado por uno o dos selectores separados entre si, el ultimo slector sera el elemento ht al que se le aplicara propiedades. css

1.7- Explicar brevemente utilizando una lista ordenada: ¿cómo funciona el CSS? Guía para respuesta

  1. paso 1: el navegador carga/recibe el html.
  2. paso 2: convierte el html en un dom, El dom representa el documento en la memoria del ordenador ( dom es como un arbol familiar cada atributo o lenguaje marcado se convierte en un nodo, que cada uno se define por su relaion con otros nodos, unos nodos pueden ser padres de otros nodos secundarios y asi su susesivamente)
  3. paso 3:el navegador va a buscar la mayor parte de los recursos vinculados al documento html, como imagenes, videos, el archivo css vilculado, etc
  4. paso 4: el navegador analiza el css y ordena en diferentes cubos las diferentes reglas segun el tipo de selector
  5. paso 5: el árbol de renderización muestra la estructura en que los nodos deben aparecer después de que las reglas sean aplicadas
  6. paso 6: finalmente se muestra el aspecto visual de la página en la pantalla

1.8- ¿Cuáles son las tres formas para vincular CSS con HTML?

las 3 formas de vincular css con html son: la vinculacion externa, utilizando la etiqueta en la seccion del documento html para referenciar un archivo css externo. los etilos en linea, utilizando la etiqueta style en las etiquetas html paras estilizar etiquetas individuales. por ultimo los estilos internos, utilizando la etiqueta en la seccion del documento html, para definir estilos css dentro del mismo archivo html.
